# reviewGOOSE:GitHub

The GitHub App connects reviewGOOSE to your repositories.

## Install

1. Visit [github.com/apps/reviewgoose](https://github.com/apps/reviewgoose)
2. Click **Install**
3. Select repositories (start with 1-2 public repos to evaluate)
4. Click **Install**

## Permissions

- **Pull requests**: Read (track PR state)
- **Contents**: Read (read `.codeGROOVE/` config files only)
- **Metadata**: Read (repo names)
- **Organization members**: Read (identify reviewers)

We do not read your source code.

## Verify

After installing, visit your [dashboard](https://reviewgoose.dev) and confirm PRs appear.

## Manage

To add/remove repos: [GitHub Settings → Installed Apps](https://github.com/settings/installations) → reviewGOOSE → Configure
