# Plans

All plans include all features. The only difference is repository access.

## Pricing

| Plan | Price | Repos |
|------|-------|-------|
| **Free** | $0 | Public only |
| **Pro** | $2/month | Public + Private |
| **Flock** | $3/seat/month | Public + Private (team billing) |

## Which Plan?

**Free**: Open source contributors. Works with any public repository.

**Pro**: Private repos for individuals. Covers work, side projects, client work.

**Flock**: Teams. Billed through GitHub Marketplace, add/remove seats anytime.

## Private Repos

1. Subscribe to Pro or Flock
2. Install the [GitHub App](https://github.com/apps/reviewgoose)
3. Select private repositories

## Enterprise

Custom contracts (MSA, DPA, BAA) or dedicated support: [flock@codegroove.dev](mailto:flock@codegroove.dev)

## FAQ

**Can I use Free and Pro together?** Yes. Free covers public, Pro covers private.

**What if I uninstall?** Data deleted within 48 hours.

**Free trial?** Use Free on public repos. The experience is identical.
